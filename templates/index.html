We have joined hands to fight with the current Covid-19 pandemic situation. We strive to help our Covid Warriors, while making it easy for a common man to get access to all the relevant information that they need at this time.

Dashboard

As we know, our Covid-warriors are fighting the pandemic very rigouresly. Our government has setup several Covid-care centers to augument the medical situation of the citizens.













To enable our citizens to view the real time data of each Covid-care centre in their locality, we have built this dashboard. This dashboard picks up data from the central database add it displays the latest data in an intuitive way.













It shows data about the active and discharged patients in each centre along with the data about the doctors and bed availibility. Every citizen can use this dashboard to select the centre that is the most convinient to them.
